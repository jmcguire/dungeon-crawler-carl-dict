Kindle install

Install the Dungeon Crawler Carl dictionary on Kindle

Download the MOBI dictionary, copy it to your Kindle over USB, then select it as an English dictionary.

Download Kindle MOBI Download Kindle bundle

Steps

  1. Download Dungeon-Crawler-Carl-Dictionary.mobi.
  2. Connect your Kindle to your computer with USB.
  3. Open the Kindle drive. On macOS it usually mounts as /Volumes/Kindle.
  4. Copy the MOBI file into documents/dictionaries/ on the Kindle.
  5. Safely eject the Kindle and unplug it.
  6. Give the Kindle a moment to index the dictionary.
  7. On the Kindle, open Settings → Language & Dictionaries → Dictionaries.
  8. Look for Dungeon Crawler Carl Dictionary under English and select it.

macOS command-line copy

If the downloaded MOBI is in your current folder and the Kindle is mounted at /Volumes/Kindle, this is the equivalent terminal copy:

cp Dungeon-Crawler-Carl-Dictionary.mobi "/Volumes/Kindle/documents/dictionaries/Dungeon_Crawler_Carl_Dictionary.mobi"

Try it

Open a Dungeon Crawler Carl book and select a name such as Carl, Donut, or Mordecai. Kindle lookup can be brittle with multiword proper nouns, so if a full phrase fails, try selecting the most distinctive single word.
