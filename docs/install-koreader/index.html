KOReader install

Install the Dungeon Crawler Carl dictionary in KOReader

Install the StarDict bundle used by KOReader's dictionary lookup system.

Download StarDict ZIP

Steps

  1. Download Dungeon-Crawler-Carl-Dictionary-StarDict.zip.
  2. Extract the ZIP.
  3. Keep the extracted Dungeon-Crawler-Carl-Dictionary folder and all of its files together.
  4. Copy that folder into koreader/data/dict/.
  5. Restart KOReader.
  6. If the dictionary is not enabled automatically, open Dictionary settings → Manage dictionaries and enable Dungeon Crawler Carl Dictionary.

Path

koreader/data/dict/Dungeon-Crawler-Carl-Dictionary/

Priority

To make it the default lookup result globally, move Dungeon Crawler Carl Dictionary above your other dictionaries in Dictionary settings → Manage dictionaries. For one book only, open that book and use Dictionary settings → Set dictionary priority for this book.

Internal links

Linked release builds use KOReader-supported bword:// links. When an entry references another known dictionary entry, tapping that reference should open the linked entry inside KOReader.
