Spoiler policy

Spoiler notes and lookup limitations

This dictionary is intended to help readers remember names and references. It is spoiler-aware, but not fully spoiler-proof.

What the dictionary does now

If a source wiki page has a page-level spoiler warning banner, the generated dictionary entry places a spoiler note above the summary. That gives readers a chance to stop before reading the definition.

What it does not do yet

The dictionary is not currently gated by book number or reading progress. The source wiki generally does not mark every spoiler phrase inside otherwise normal sentences, so a page-level warning is the safest available signal.

How to use it carefully

If you are a first-time reader, stop when you see a spoiler warning. If you are especially spoiler-sensitive, use the dictionary mainly for names you already recognize or after finishing the book where the name appears.

Possible future improvement

A later version could create separate book-specific dictionaries, such as a Book 1 dictionary, Book 2 dictionary, and so on. That would require reliable source data for first appearance and spoiler boundaries.
