Kobo install

Install the Dungeon Crawler Carl dictionary on Kobo

Copy the custom Kobo dictionary ZIP into your Kobo's custom dictionary folder.

Download dicthtml-dc.zip Download Kobo bundle

Steps

  1. Download dicthtml-dc.zip.
  2. Connect your Kobo to your computer with USB.
  3. Open the Kobo drive, usually named KOBOeReader.
  4. Copy dicthtml-dc.zip into .kobo/custom-dict/.
  5. If custom-dict does not exist, create it inside .kobo.
  6. Safely eject and restart the Kobo.
  7. Open a book, select a word, and open the dictionary panel.
  8. Use the dictionary selector to choose the custom dictionary for the dc locale.

Path

KOBOeReader/.kobo/custom-dict/dicthtml-dc.zip

Firmware note

Current Kobo firmware supports custom dictionaries in .kobo/custom-dict/. Older firmware may require ExtraLocales or a custom dictionary patch before the dictionary can be selected.
