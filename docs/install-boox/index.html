BOOX install

Install the Dungeon Crawler Carl dictionary on BOOX

Use the StarDict bundle on a BOOX e-reader that supports StarDict dictionary files.

Download StarDict ZIP

Steps

  1. Download Dungeon-Crawler-Carl-Dictionary-StarDict.zip.
  2. Extract the ZIP.
  3. Keep the extracted Dungeon-Crawler-Carl-Dictionary folder and all of its files together.
  4. Import that folder into your BOOX dictionary library, or copy it into the location BOOX uses for StarDict dictionaries.
  5. Refresh or restart the reader app if the dictionary does not appear right away.
  6. If needed, open your dictionary settings and set Dungeon Crawler Carl Dictionary as the preferred English dictionary.

What is in the folder

The BOOX build uses the same StarDict bundle as KOReader. Keep the .ifo, .idx, .dict, .syn, and .css files together inside the extracted dictionary folder.

Priority

BOOX dictionary menus vary a bit by app and firmware, but the goal is the same: enable Dungeon Crawler Carl Dictionary and move it ahead of other English dictionaries if BOOX lets you set dictionary priority.

Internal links

Linked release builds include internal entry links in the StarDict definition HTML. Support for following those links depends on the BOOX dictionary app you use.
