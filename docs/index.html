What this is

This project builds Kindle, StarDict for KOReader and BOOX, and Kobo lookup dictionaries from pages on a MediaWiki/Fandom wiki. The current default target is the Dungeon Crawler Carl Fandom character category, and the toolchain is designed so more categories or other Fandom wikis can be added later.

The goal is practical reading help: select a term such as Carl, Donut, or Mordecai while reading and get a short in-device lookup entry.

Start here

Install on Kindle

Copy the MOBI file to the Kindle dictionaries folder and select it under English dictionaries.

Install on Kobo

Copy the custom dictionary ZIP into .kobo/custom-dict/.

Install in KOReader

Extract the StarDict bundle and put the folder under koreader/data/dict/.

Install on BOOX

Extract the StarDict bundle and import or copy it into your BOOX dictionary library.

Reader-friendly

Built for lookup while reading, not as a replacement for the full wiki. Entries are intentionally short.

Spoiler-aware, not spoiler-proof

Entries preserve page-level spoiler warnings from the source wiki when available. The dictionary is not yet fully book-by-book spoiler-gated.

Known lookup limitations

Kindle and other e-reader lookup interfaces can be picky about proper nouns and multiword phrases. If a full phrase does not trigger a useful result, try selecting the distinctive single word in the phrase. Some Kindle internal links work when opening the dictionary as a book but may not work inside the lookup card UI.
